How It Works

This calculator approximates the volume of the fish model using Riemann sums. The volume is calculated by:

  1. Taking cross-sections perpendicular to the x-axis
  2. Using the difference between f(x) and g(x) as the diameter of semicircles
  3. Calculating the area of each semicircle: A = π/8 × (difference)²
  4. Summing up all the areas multiplied by the width of each rectangle (Δx)

The formula for volume: V = π/8 × ∫(upper function - lower function)² dx
